# Práce s GitHubem

## 1. Vytvoření nového repozitáře
1. Přihlaste se ke svému účtu na [GitHub.com](https://github.com/).
2. V pravém horním rohu klikněte na ikonu **+** a zvolte **New repository**.
3. **Repository name:** Zvolte název (např. `napad-ukol`).
4. **Public/Private:** Povinně zvolte možnost **Private** (Soukromý).
5. Zaškrtněte políčko **Add a README file**.
6. Klikněte na tlačítko **Create repository**.

## 2. Vytvoření dokumentace a kódu
Všechny soubory vytvořte přímo v prohlížeči pomocí tlačítka **Add file** -> **Create new file**.

### Návod (.md)
* Pojmenujte soubor např. `navod.md`.
* Vložte krátký text popisující vaši práci.
* Pro uložení klikněte na zelené tlačítko **Commit changes...** (potvrďte uložení).

### Cvičení (.py)
* Znovu klikněte na **Add file** -> **Create new file**.
* Pojmenujte soubor např. `souhvezdi.py`.
* Vložte svůj vypracovaný kód v Pythonu.
* Klikněte na **Commit changes...** a potvrďte uložení.

## 3. Nasdílení vyučujícímu (Přidání spolupracovníka)
Protože je váš repozitář soukromý, musíte vyučujícímu udělit přístup, aby mohl úkol zkontrolovat:

1. V horním menu svého repozitáře klikněte na **Settings** (ikona ozubeného kolečka).
2. V levém panelu zvolte sekci **Collaborators** (pod záložkou *Access*).
3. Klikněte na tlačítko **Add people**.
4. Do vyhledávacího pole zadejte uživatelské jméno vyučujícího: **[ZDE DOPLŇTE VAŠE GITHUB JMÉNO]**.
5. Klikněte na **Add [jméno] to this repository**.

*Poznámka: Vyučujícímu přijde pozvánka e-mailem, kterou musí potvrdit, aby váš kód viděl.*
